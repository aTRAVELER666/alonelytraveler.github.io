实现方法

1. Pyodide (浏览器内Python)

Pyodide将Python编译为WebAssembly，使其能在浏览器中运行。

<script src="https://cdn.jsdelivr.net/pyodide/v0.23.4/full/pyodide.js"></script>
<script>
  async function main() {
    let pyodide = await loadPyodide();
    return pyodide.runPython(`
      import sys
      sys.version
    `);
  }
  main().then(version => {
    console.log(version);
  });
</script>

2. 服务器端Python

使用Flask、Django等框架在服务器运行Python，通过API与前端交互。

from flask import Flask, request, jsonify

app = Flask(__name__)

@app.route('/run-python', methods=['POST'])
def run_python():
    code = request.json.get('code')
    try:
        # 注意：在生产环境中执行用户代码需要严格的安全措施
        result = exec(code)
        return jsonify({'result': str(result)})
    except Exception as e:
        return jsonify({'error': str(e)})

if __name__ == '__main__':
    app.run(debug=True)

3. 转换为JavaScript

使用Transcrypt等工具将Python代码转换为JavaScript。

# Python代码
def calculate(a, b):
    return a * b + 10

# 转换为JavaScript后
function calculate(a, b) {
    return a * b + 10;
}
注意：在浏览器中直接运行Python代码有一定的性能限制，且无法使用所有Python库。对于复杂应用，建议使用服务器端方案。

更多信息

要在生产环境中使用Python与HTML/CSS/JavaScript结合，可以考虑以下方案：

  • Web框架：使用Django、Flask或FastAPI创建Web应用
  • Jupyter Notebook：在网页中交互式运行Python代码
  • Brython：另一种在浏览器中运行Python的替代方案
  • PyScript：一个新框架，允许在HTML中嵌入Python
